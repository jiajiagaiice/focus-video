# Roadmap

## M0：工程骨架

- 建立 C++20 / CMake 项目。
- 定义视频参数、目标输出分辨率、GPU 能力和超分档位选择逻辑。
- 给 3060 Ti、2K/4K 输出目标和降级路径写单元测试。

## M1：可播放本地文件

- 集成 FFmpeg demux / decode。
- Windows 上建立 D3D11 渲染窗口。
- 支持播放、暂停、拖动、音量和基础快捷键。
- 建立播放指标面板：FPS、帧耗时、队列深度和丢帧数。

## M2：GPU 零拷贝链路

- 接入 NVDEC 硬解。
- 建立 GPU frame pool。
- 减少 CPU 回读，验证 1080p60 原始播放稳定性。

## M3：实时超分 MVP

- 接入 TensorRT FP16 推理。
- 支持 2x Performance / Balanced / Quality 三档模型。
- 在 3060 Ti 上调优 1080p30 -> 2K/4K30 Balanced 档。
- 增加一键开关和自动降级策略。

## M4：播放器体验完善

- 字幕渲染和音轨选择。
- 记忆播放进度和每视频超分配置。
- 崩溃日志、性能日志和用户可导出的诊断包。

## M5：发布准备

- Windows 安装包。
- GPU / 驱动版本兼容性检查。
- 首次启动性能基准测试，自动选择默认超分档位。
